## Installation - ReactJS
 
| **Tools Used** | **Version** |
| -------------- | ----------- |
| ReactJS        | 18.2.0      |
| Material UI    | 5.14.1      |
| Vite           | 4.4.5       |
| VS Code        | IDE         |

- #### Running Project
  
```cd mozan```<br>
```npm install```<br>
```npm run dev```
